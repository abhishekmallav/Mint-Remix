# Mint-Remix

## Mint Remix is a customization script for Linux Mint

To install just run the script

```bash
chmod +x script.sh
./script.sh
```

#### Installed Themes

- WhiteSur GTK Theme

- Fluent-Compact GTK Theme

- Graphite GTK Theme

- Lavanda-Sea GTK Theme

- Otis-Forest GTK Theme

- Adara-Dark Theme

- Tela-Circle Icon Theme

- WhiteSur Icon Theme
